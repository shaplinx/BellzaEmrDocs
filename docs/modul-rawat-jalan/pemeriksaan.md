# Pemeriksaan
**Fungsi**: Menginput data hasil pemeriksaan pasien.

## Fields Pemeriksaan:
- Anamnesis
- Chief Complaint SCT
- Vital Signs:
  - Tekanan Darah
  - Kesadaran
  - Keadaan Umum
  - Laju Nadi
  - Laju Nafas
  - Suhu
  - Saturasi
- Antropometri:
  - Berat Badan
  - Tinggi Badan
- Buta Warna
- Pemeriksaan Fisik:
  - Lokasi
  - Temuan
- Status
- Diagnosis:
  - Nama
  - ICD-10
